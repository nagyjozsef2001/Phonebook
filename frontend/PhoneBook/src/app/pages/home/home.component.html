Welcome to Our Phonebook

A minimalistic phonebook web application

About the Project

This project's frontend was written in Angular, featuring a minimalistic design and a powerful searchbar for searching by name or phone number. It serves as a phonebook web application and supports CRUD operations such as creating contacts and user accounts, login functionality, and modifying or deleting contacts. The backend of the page was developed using Java, Spring Boot, and Spring Security to ensure a secure API.
